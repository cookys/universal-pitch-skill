## [0.0.1] - 2026-04-08

### Added

- 4-phase multi-agent pipeline: Scanner → UX Architect → Marketing Pro → Review Gate
- Phase 1: Project Scanner with structured JSON output contract
- Phase 2: JTBD 3-layer analysis (functional/social/emotional) + Seibel's 7 Questions validation
- Phase 3: 3-layer selling point distillation (feature → capability → emotion)
- Phase 3: Pitch deck structure (YC-influenced, 10-18 slides by round)
- Phase 3: Landing page trust-building curve structure
- Phase 3: Visual strategy branching for GUI vs non-UI products
- Phase 3: Screenshot decision tree (job-based: category / value proof / wow moment)
- Phase 3: Input → Magic → Output formula for CLI/API/infrastructure products
- Phase 4: 8-point review gate checklist
- Reference: YC 7 Questions, JTBD template, Value Proposition Canvas, Mom Test rules
- Reference: Playwright screenshot automation templates + common pitfalls
- Reference: Non-UI visual assets guide (VHS, Carbon, D2, Swark, before/after flows)
- Anti-patterns guide with fixes
- Audience-specific messaging matrix (investors vs customers)
- Competition positioning guide (2x2 matrix)
